Norse Runes API Documentation

The Norse Runes API is an easy to use and free API that serves data on Norse runes. It is similar to Tarot APIs.

Finding a specific rune

URL path: /api/find

https://www.example.com/api/find?query=value
Query Acceptable values
rune any rune name from the following: fehu, uruz, thurisaz, ansuz, raidho, kenaz, gebo, wunjo, hagalaz, nauthiz, isa, jera, eihwaz, perthro, algiz, sowilo, tiwaz, berkana, ehwaz, mannaz, laguz, ingwaz, othala, dagaz, odin

Drawing runes

URL path: /api/draw

https://www.example.com/api/draw?query=value
Query Acceptable values
amount any numerical value between 1-25
